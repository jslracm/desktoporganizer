Editing "Comments"
User's Guide > Edition of a configuration file > Editing "Comments"

Both Groups and Options may have associated Comments. The Comment is edited using the following form:

Comment text allows you to include certain HTML elements, such as:

  • Text with font bold, italic or underlined.
  • Hyperlinks, to external (Internet) URLs of to locations in the file system.
  • Images.
  • etc.

The list of allowed HTML elements are detailed here.

In general, these elements are specified in normal text pairs enclosed in "<>". For example, "<p> any text .... </ p>" appears (HTML) so that "any text ...." appears in a new paragraph. Others, such as the insertion of images, or a hyperlink, have a more complex structure. To aid in the insertion of these elements, the "Edit the comment ..." form includes a menu with the options shown in the following figure.

The general procedure to use the options of this menu "Insert a new paragraph".

Rules for Comments.

Comments are HTML texts that support a subset of HTML elements.

  • Insert a paragraph.
  • Change the Font of part of a text.
  • Insert an image.
  • Insert a hyperlink.
