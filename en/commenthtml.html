HTML elements supported
User's Guide > Edition of a configuration file > Editing "Comments" > HTML elements supported

The HTML elements allowed in the Commentary texts are listed below. Refer to w3schools.com full description and method of use of these items.

Element

Description

Associated attributes

a

Defines a hyperlink.

href, id, name, title

b

Defines bold text.

big

Defines big text (obsolete).

blockquote

Defines a section that is quoted from another source.

body

Defines the document's body.

background, bgcolor, text, link

br

Defines a single line break.

center

Defines centered text.

em

Defines emphasized text.

font

Defines font, color, and size for text.

face, color, size

h1 a h6

Defines HTML headings.

hr

Defines a thematic change in the content.

i

Defines a part of text in an alternate voice or mood. (Also italic text font).

img

Defines an image.

alt, border, height, hspace, src, vspace, width
NOTE: images should be in ".jpg" format.

li

Defines a list item.

ol

Defines an ordered list.

p

Defines a paragraph

New paragraph

small

Defines smaller text.

strong

Defines important text

table

Defines a table.

align (left and center), border, cellpadding, cellspacing, width

thead

Groups the header content in a table.

tbody

Groups the body content in a table.

tfoot

Groups the footer content in a table.

tr

Defines a row in a table.

th

Defines a header cell in a table.

align, colspan, width

td

Defines a cell in a table.

align, colspan, width

u

Defines text that should be stylistically different from normal text. (Also underline text font).

ul

Defines an unordered list.
