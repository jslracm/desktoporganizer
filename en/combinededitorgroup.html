Groups: create, edit, delete
User's Guide > Advanced subjects > Combined editor for Groups and Options > Groups: create, edit, delete

The Groups part of the Combined Editor of Groups and Options form contains the following components.

The Existing and New check boxes (top left) allow  to decide if what you want to edit one of the existing Groups or to create a new one.

If the selected Group (the one shown it the combobox associated to the Existing checkbox) contains one or more Options, the combobox associated to Options in the selected Group is populated with these Options, and the first one is selected and its data are used to populate the Options part of the form.

The green button labeled "new Option" fill the Options part of the form with blank values (reset).

If the selected Option changes the Options part is fulfil with the content of the newly selected one.
