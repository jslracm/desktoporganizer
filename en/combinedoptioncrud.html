Options: create, edit, delete
User's Guide > Advanced subjects > Combined editor for Groups and Options > Options: create, edit, delete

The Options part of the "Combined editor for Groups and Options" is used to create and edit Options associated to the selected Group in the Groups part. Its layout is shown in the next figure.

When there is a change in the "Options in the selected Group" combobox, the data of the selected Option (shown) are used to fill it the Options part, and the background color is changed to pink color. See below.

New Option

You can create a "New Option" associated to the selected Group in the Groups part, pressing the "new Option" green button. In this case the Options part background changes to green (see below).

You must fulfil at least the Command and the Title textboxes, and the Command must be tested with the "Command Test" button (with a "Yes" response). After that you may press "Add Option" to actually create and add the new Option to the selected Group.

The Command textbox can be used for dragging & dropping files on it (see here). The "Advance command editor" presents the "Enter command and parameters" form.

The Comment can be changed bringing up the "Edit the Commentary ..." form clicking with the mouse right button on the Comment panel. See possibilities here.

Tags can be changed directly typing in the "Tags" textbox. Also, double-clicking on any of the items in the "Existing tags" listbox its value is copied to the Tags textbox.

The "Move to Group" button (bottom right of the form), can be used to move the existing Option to the Group selected in the associated combobox.
