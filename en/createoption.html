Creating options
User's Guide > Edition of a configuration file > Options: create, edit, delete > Creating options

The Options of a Group constitute the shortcuts to documents or applications. Desktop Organizer allows creating them very easily, by various methods.

  1. Using the Drag & Drop functionality.
  2. With the "Add Option" menu option of the Group menu.
  3. Using the Combined editor for Groups and Options.
