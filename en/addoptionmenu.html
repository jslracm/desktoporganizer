Add a new Option with the Group menu.

An Option can be created using the Add Option menu of the Group menu.

Pressing the left button on this "Add Option" presents the following form:

The Command text box allows to "drag and drop" a file on it.

NOTE: You will see that in the "Existing tags" list, the tags that are already associated with an Option appear. If the text in Tags contains duplicated tags, they will be cleaned up after pressing Save data and return.
