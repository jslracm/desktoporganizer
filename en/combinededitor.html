Combined editor for Groups and Options
User's Guide > Advanced subjects > Combined editor for Groups and Options

The form Combined Groups and Options editor  is designed to facilitate the creation and edition of many Groups and Options at once. Its layout is shown in the next figure.

It contains two parts: one for Groups and another for Options.

This form is presented as result of several actions:

  1. Using the Edit loaded configuration file option of the Desktop Organizer files menu.

If the configuration file contains at least one Group, this editor is opened selecting the first one (alphanumerically ordered). See next figure.

  1. Using the Create a new configuration file" option of the "Desktop Organizer files" menu.
    In this case the form is created as shown in the next figure:

In this case, the first action must be to create a "Group".
