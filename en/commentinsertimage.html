Insert an image
User's Guide > Edition of a configuration file > Editing "Comments" > Insert an image

Adding an image to HTML text requires specifying several parameters: to facilitate this work, the form changes by adding additional fields or functionalities.

First, place the cursor of the text box in the position where you want to insert the image. Use the menu option Add an image and you will see that the form changes by adding the fields shown in the figure below:

It is necessary to "drag and drop" the image file that you want to insert into the "Image location" field. The image must be in JPG format. The "Image width" and "Image height" fields are automatically filled in with the actual values of the image in the file. The image may have dimensions that are suitable for viewing in the Comment: you can change the size with which it is presented by changing the value of the "Image width" and "Image height" text boxes. If the "Maintain proportions" box is kept checked, if the "Image width" is changed, "Image high" is modified accordingly to maintain the shape factor. This way you can adapt the size of the presentation of the image in the Commentary. Image measures are in "pixels".

It is also necessary to fill in the field "Alternative text".

"Test"button  must be pressed so that the modified text appears in the text block of the form and also in the result is presented in the "Comment" of the Standard View. Next figure shows an example:
