Access to documents through tags
Objectives of this application > Access to documents through tags

The Desktop Organizer uses the concept of "Tag" to facilitate the creation of classes of Options. The tags may have more than one level, they may contain a hierarchy. To separate levels a tag uses the character ":". Next table shows a number of examples.

Example

Description

"novel"

First level tag: "novel".

"novel:historic"

This expression contains two tags:
a) a first level tag : "novel", and
b) a second level tag: "historic"

"historic"

This expression contains a single, first level tag: "historic".
Note that this is different from "novel:historic"

"historic novel"

This expression contains a single, first level tag: "historic novel".
Note that this is different from "novel:historic"

"Tag Tree View" shows how to access Options through tags.

"Search View" shows how to narrow searches using tags.
