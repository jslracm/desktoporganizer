2.5 Acceso a documentos mediante etiquetas
Objetivos de esta aplicación > Acceso a documentos mediante etiquetas

Una Etiqueta es un texto cualquiera, que puede contener cualquier carácter los siguientes ":"'". El carácter ":" se utiliza para introducir jerarquía en una etiqueta.

Ejemplos:

Ejemplo de etiqueta

Descripción

"novela"

Etiqueta de primer nivel

"novela:historica"

Esta expresión se refiere a dos etiquetas: "novela" e "historica"; esta última existe como secundaria de "novela"

"historica"

Esta expresión se refiere a una etiqueta de primer nivel, diferente de "novela:historica"

"novela historica"

Se refiere a una etiqueta de de primer nivel, que contiene espacios en blanco y es  diferente de "novela:historica"

El uso principal de las Etiquetas es poder clasificar las Opciones fuera de los Grupos en los que están definidas. Así, los Grupos actúan como "listas de lectura asociado a un criterio particular", y las Etiquetas permiten localizar Opciones de acuerdo con criterios más genéricos.

En Vista por arbol de etiquetas se muestra como acceder a las Opciones a través de Etiquetas.
