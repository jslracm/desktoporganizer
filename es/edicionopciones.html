4.4.2 Opciones : Creación, edicion y eliminación
Guía de Usuario > Edición de un fichero de configuración > Opciones : Creación, edicion y eliminación
  • Creación de Opciones. Esté epigrafe describe las diversas modalidades para la creación de Opciones
  • Edición de detalles de Opciones

A continuación se describen las diferentes opciones para cambiar las propiedades de una Opción.

En la Vista estandar y pulsando el botón derecho del ratón cuando está sobre la Opción a editar, se muestra el siguiente menú.

A continuación se describen las diferentes funcionalidades accesibles a través de este menú.

    • Acceso al editor de Detalles de la Opción

Esta opción presenta el "Editor combinado de Grupos y Opciones", rellenado con las propiedades de la Opción seleccionada. En este epígrafe se describen las funciones disponibles en este Editor.

    • Cambiar el Comentario: El comentario es un texto HTML. El editor descrito en "Edición del Comentario" le ayuda en su edición.
    • Cambiar Comando.

El Comando se refiere al fichero, URL o aplicación asociados a la Opción, que es ejecutado cuando se hace doble-click sobre ella. Es decir, presenta el documento indicado utilizando la aplicación asociada a su extensión ("*.ext") (para los documentos), o abre una ventana del navegador de Internet con la URL (caso de las URLs), o ejecuta la aplicación con los parámetros que tenga asociados en el Comando (en el caso de las aplicaciones).

Esta opción del menú presenta el siguiente formulario, en donde se puede cambiar:

      • Cambiar el documento a abrir, o
      • Introducir una URL, o
      • Especificar la ubicación la aplicación deseada, incluyendo la especificación de los parámetros que necesite o se desen. Ver también "Asociar parámetros al Comando".

      • Cambiar Etiquetas: Ver Edición de las Etiquetas de una Opción.
      • Cambiar Título. Permite cambiar el Título que se presenta de la Opción, a través del formulario que se muestra en la figura siguiente.

      • Cambiar fuente del Título.

Esta opción del menú  se usa para resaltar una determinada Opción, cambiando el estilo de su Título: su Fuente, Estilo, Tamaño, Efectos y Color del Título de la Opción.

      • Mover la Opción a otro Grupo.

Esta opción del menú presenta el siguiente formulario, donde se puede elegir un Grupo, al que se moverá la Opción seleccionada.

      • Borrar esta Opción.

Para eliminar la Opción seleccionada.

      • Mover el archivo de Comando a otra ubicación.

Esta opción permite cambiar la ubicación del documento asociado a la Opción. Presenta el formulario para poder elegir la nueva ubicación.

NOTA: Cuando se quiere cambiar la ubicación del fichero a que hace referencia una Opción, DEBE USAR ESTA FUNCIONALIDAD, ya que, si lo hace manualmente, la Opción dejará de ejecutar su funcionalidad cuando se haga doble-click sobre ella.
Esta funcionalidad es equivalente a mover manualmente el fichero de referencia y cambiar el "Comando" de la Opción para que apunte a la nueva ubicación.
