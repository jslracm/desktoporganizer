4.4.3 Edición del Comentario
Guía de Usuario > Edición de un fichero de configuración > Edición del Comentario

Tanto los Grupos como las Opciones pueden tener Comentarios asociados.

La edición de un Comentario se realiza mediante el formulario siguiente:

El texto del Comentario permite incluir determinados elementos HTML, tales como:

  • Palabras con texto en negrita, o cursiva, o subrayadas.
  • Enlaces a direcciones de Internet o a ficheros en el propio PC (hyperlink).
  • Imágenes.
  • etc.

El listado de los elementos HTML permitidos se detallan aquí.

En general, estos elementos se concretan en parejas de texto normal encerradas en "<>". Por ejemplo, "<p> Texto cualquiera .... </p>" se presenta (HTML) de forma que "Texto cualquiera ...." aparece en un nuevo párrafo. Otros, como la inserción de imágenes, o de un hiperenlace, tienen una estructura algo más compleja. Para ayudar en la inserción de estos elementos, el formulario "Edite el Comentario ..." incluye un menú con las opciones que se muestran en la siguiente figura.


Normas para los Comentarios.

Los comentarios son textos HTML que admiten un subconjunto de elementos HTML.

  • Insertar un párrafo.
  • Cambiar el tipo de carácter de una parte del texto.
  • Insertar una imagen.
